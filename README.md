# Simply Books

React + Vite frontend for Software Application Domain project.

## Tech Stack
- React
- Vite
- Neon
- Express
- Node.js
- GitHub Pages

## Setup
1. Clone this repo onto your local machine
2. type 'npm install' in the terminal with this project open
3. type 'npm run dev' and click the localhost link to see the website while editing
4. in a new terminal (DO NOT CLOSE YOUR OLD ONE), type 'cd backend' and 'node index.js' to connect to the database
5. Any changes made to the backend while running, you MUST ctrl+c in the terminal and run the commands in step 4 again
